# Fractal Explorer

## Project Purpose

This project contains a development version of a fractal renderer/explorer. It is designed as a space to adjust as many parameters as possible when resolving and rendering fractals. Some version of it will eventually serve as the engine for an Idle Fractal game.The current design document for the game can be found in`FractalIdleMDD.md`. The game will have a separate engine for implementation of standard idle game functions; this engine will be used to generate the fractals and provide the player with a visual representation of their progress.

## Instructions

`FractalIdleMDD.md` is the current and most up to date design document for the game. It is a work in progress and will be updated. Do not take it to be immutable or gospel truth; however, when in doubt, use it as a reference for the current state of the game design. The game is still in development and the document will be updated as new features are added and old features are modified or removed. If the document need to be modified, that should be flagged and discussed.
