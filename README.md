# Mandelbrot Scratchpad

A minimal standalone Mandelbrot renderer built in TypeScript + Canvas for performance testing.

## Run locally

```bash
npm install
npm run dev
```

Then open the local Vite URL in your browser.

## Controls

- Drag to pan.
- Scroll to zoom.
- Click to zoom in; right-click to zoom out.
- Adjust canvas size, max iterations, and tile size from the sidebar.
- Use the Render button to re-render the current view.
